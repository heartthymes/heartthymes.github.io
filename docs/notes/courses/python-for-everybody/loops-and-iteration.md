 ## Loops and iteration
 
>Date: 2023-03-05  
>Category: [00ComputerScience](links/00ComputerScience.md)  
>Tags: #python #programming  

While loop  
```python
while x > 0:
	#statements
	break #exits the loop
	continue #skips current iteration and proceeds to the next one
```

For loop
```python
for i in range(5):
	#statements

for i in ["one", "two"]:
	#statements
```

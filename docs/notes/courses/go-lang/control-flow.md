## Control flow
 
>Date: 2023-03-29  
>Category: [00ComputerScience](links/00ComputerScience.md)  
>Tags: #golang #programming  

```go
x := true

if x {
	//some code
} 

for i:=0; i<10; i++ {
	//some code
}

i = 0
for i < 10 {
	//some code
	i++
}

for {
	//some code
}

switch x {
	case x == 1:
		//some code
	case x == 2:
		//some code
	default:
		//some code
}
```

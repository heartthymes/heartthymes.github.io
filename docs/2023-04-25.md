## Rust

### Installation
To install  
```bash
curl --proto '=https' --tlsv1.3 https://sh.rustup.rs -sSf | sh
```

To check  
```bash
rustc --version
```

To update
```bash
rustup update
```

To uninstall
```bash
rustup self uninstall
```

To view documentation locally
```basg
rustup doc
```
